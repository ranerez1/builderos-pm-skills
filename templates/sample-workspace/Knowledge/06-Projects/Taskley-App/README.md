# Taskley

A simple task list in **one HTML file**. Double-click `index.html` to open it—no Node, no install, no server.

Tasks are saved in your browser (localStorage).

---

## How to use

1. Open **`index.html`** in Chrome, Safari, Firefox, or Edge.
2. Add tasks, set priority (P1–P3), due dates, and notes.
3. Your list persists when you close and reopen the file (same browser).

---

## Features

- Add, edit, complete, and delete tasks
- Priorities, due dates, notes
- Filter: All / Active / Completed
- Clear completed tasks

---

## Notes

- Data stays on **this device** in **this browser**. Clearing site data removes your tasks.
- Copy the whole `Taskley-App` folder (or just `index.html`) to share—the file is self-contained.
